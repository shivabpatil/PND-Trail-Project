You can create separate folder for services inside server folder

Models for login can be included inside the models folder in side server. We have created few blank models already
you can use them or create your own.

Routes related to the login can be added in side the Routes folder inside server older.

All the initial configurations for passport can be added to express.js inside config folder in side server folder.
You will find few configuration already there. You can change them if needed.


All client side code for and views can be added to account folder inside public/app
